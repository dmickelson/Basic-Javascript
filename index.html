Variables

Variables are case sensitive. Can only contain with letters, numers, underscores, dollar signs. Can begin with a letter, _ and $.

For each loops

foreach loops are designed to work with arrays.

var numbers = [33,54,76,34,2,6]; numbers.foreach(function(number) { console.log(number); });

Output:

33 54 76 34 2 6

This is equivalent to:

var numbers = [33,54,76,34,2,6]; numbers.foreach(function(number) { console.log(number); });

Output:

33 54 76 34 2 6
